# Changelog

All notable changes to ClipStack will be documented here.

---

## [0.1.0] — 2026-04-03

Initial public release.

### Features
- Automatic clipboard capture with persistent SQLite history
- Instant fuzzy search across all clips
- Pin clips to keep them at the top
- Drag-to-reorder pinned clips
- Global keyboard shortcut to open the overlay (`Alt+Shift+V` by default, fully rebindable)
- Keyboard navigation — arrow keys + Enter to copy without a mouse
- Image support — captures and previews copied images
- Dark / light / system theme
- Launch at startup
- History export to `history.json` for manual viewing and editing
- Minimal resource footprint — < 30 MB RAM, < 0.1% CPU when idle
